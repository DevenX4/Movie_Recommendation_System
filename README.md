# Movie Recommendation System

**Technologies:** Python, Streamlit, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

---

## Project Overview
This interactive movie recommendation system allows users to explore movie datasets and receive personalized movie suggestions.

---

## Features
- **Interactive Interface:** Built with Streamlit for user-friendly exploration.
- **Exploratory Data Analysis (EDA):** Understand genre distribution, rating patterns, and user preferences.
- **Data Preprocessing:** Handles missing values, feature extraction, and normalization using Pandas and NumPy.
- **Content-Based Recommendations:** Suggests movies based on similarity measures and user input.
- **Visual Insights:** Charts, heatmaps, and distribution plots support data-driven recommendations.
